# Lynxz OhMyPosh Theme

Custom OhMyPosh theme for Fish shell.


(if you wanna see the ohmyposh theme you can check https://github.com/Lynxz411/lynxz-fastfetch) 

## Install

```auto
bash -c "$(curl -fsSL https://raw.githubusercontent.com/Lynxz411/Lynxz-ohmyposh/main/install.sh)"
```
## Or use this

```bash
git clone https://github.com/Lynxz411/ohmyposh.git
cd Lynxz-ohmyposh
chmod +x install.sh
./install.sh
